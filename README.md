# Kaggle Projects

個人のKaggleコンペティション・データセット分析用リポジトリ

## ディレクトリ構成

- `competitions/` - 参加中のコンペティション
- `datasets/` - 共通データセット
- `experiments/` - 実験・検証用コード
- `docs/` - ドキュメント・ノート

## 使い方

新しいコンペに参加する場合：
```bash
cd competitions
mkdir competition-name
cd competition-name
mkdir {data,notebooks,src,submissions}
```
